# School-MM-2026-GlavStroy
Кейс №3 — Школа математического моделирования ИТМО. GAN + NEAT+BNN для прогнозирования свойств бетона. Предсказание прочности и удобоукладываемости по составу смеси. Работа с малыми выборками, учёт неопределённости и физических ограничений.

## Задача

Необходимо разработать модель, которая по составу бетонной смеси предсказывает её физико-механические свойства, прежде всего **развитие прочности во времени**.

## Предлагаемый подход

Решение предлагается реализовать с помощью **GAN** (состязательного алгоритма):

- **Генератор** получает на вход состав смеси и формирует прогноз её свойств.
- **Дискриминатор** оценивает реалистичность и физическую правдоподобность предсказания.

В качестве дискриминатора должен быть использован один из вариантов **NEAT+BNN** (участникам будет предложено три реализации в качестве отправной точки).

Особое внимание стоит уделить:
- работе с **малыми выборками**,
- **устойчивости модели**,
- возможности **учитывать неопределённость прогноза**.

## Формальная постановка

**Входные данные:**
Состав бетонной смеси с известными компонентами, например:
- цемент,
- вода,
- песок,
- щебень,
- минеральные добавки,
- пластификатор.

Каждый признак измеряется в **кг/м³**.

**Выходные данные (целевые свойства):**
- **Прочность на сжатие** (измеряется в МПа) на разных сроках твердения.

**Условия:**
- Свойства смеси зависят от состава **нелинейно**.
- Данные **ограничены по объёму**.

В рамках задачи предлагается исследовать **GAN-подход**, где:
- генератор предсказывает свойства по составу,
- дискриминатор на базе NEAT+BNN проверяет, насколько эти предсказания согласуются с лабораторными данными и физическими ограничениями.

## Цель

Получить модель прогноза свойств смеси с **как можно меньшей ошибкой** по отношению к экспериментальным данным.

Важно, чтобы модель:
- точно предсказывала значения прочности на различных сроках,
- оставалась **устойчивой на малых выборках**,
- корректно работала на **новых составах**,
- давала **оценку неопределённости** предсказания.

## Бонусы (дополнительные требования)

1. **Перенос обучения** на более узкую лабораторную выборку, где диапазон составов и свойств ограничен сильнее, чем в основной базе.
2. Решения, которые **устойчиво работают на малых данных** и демонстрируют хорошую обобщающую способность.
3. **Прогнозирование прочности в зависимости от времени.**
4. **Прогнозирование нескольких свойств** (например, удобоукладываемости) бетонной смеси (для этого параметра зависимости от времени нет).

## Метрики для сравнения решений

- MAE
- RMSE
- MAPE
- Коэффициент детерминации R²

## Данные

В качестве отправной точки будет предоставлен набор данных с:
- лабораторными наблюдениями по составам бетонных смесей,
- развитием прочности во времени,
- связанными нормативными ограничениями из ГОСТ.

На основе этих данных участникам необходимо построить обучающую выборку для генератора и дискриминатора.

### Особенности данных

В силу производственных процессов измерения признаков и прочности на сжатие **подвержены зашумлению**.
